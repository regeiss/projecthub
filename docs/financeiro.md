● CPI — Cost Performance Index (Índice de Desempenho de Custo)

CPI = EV ÷ AC

- EV (Earned Value): quanto do orçamento "foi ganho" com base no trabalho concluído
- AC (Actual Cost): quanto foi realmente gasto

┌─────────┬─────────────────────────────────────────────────────────────────┐
│ Valor │ Significado │
├─────────┼─────────────────────────────────────────────────────────────────┤
│ CPI > 1 │ Abaixo do orçamento — cada R$1 gasto gerou mais de R$1 de valor │
├─────────┼─────────────────────────────────────────────────────────────────┤
│ CPI = 1 │ Exatamente no orçamento │
├─────────┼─────────────────────────────────────────────────────────────────┤
│ CPI < 1 │ Acima do orçamento — cada R$1 gasto gerou menos de R$1 de valor │
└─────────┴─────────────────────────────────────────────────────────────────┘

Exemplo: CPI = 0.85 significa que para cada R$1,00 gasto, apenas R$0,85 de trabalho foi entregue.

---

SPI — Schedule Performance Index (Índice de Desempenho de Prazo)

SPI = EV ÷ PV

- PV (Planned Value): quanto do orçamento deveria ter sido consumido até hoje, pelo cronograma

┌─────────┬───────────────────────────────────────────────────────────────────────┐
│ Valor │ Significado │
├─────────┼───────────────────────────────────────────────────────────────────────┤
│ SPI > 1 │ Adiantado — entregou mais do que o planejado para este ponto no tempo │
├─────────┼───────────────────────────────────────────────────────────────────────┤
│ SPI = 1 │ No prazo │
├─────────┼───────────────────────────────────────────────────────────────────────┤
│ SPI < 1 │ Atrasado — entregou menos do que o planejado para este ponto no tempo │
└─────────┴───────────────────────────────────────────────────────────────────────┘

Exemplo: SPI = 0.70 significa que o projeto entregou apenas 70% do que deveria ter entregado até hoje.

---

No contexto do ProjectHub, o PV é calculado como orçamento × % do tempo decorrido e o EV como orçamento × % de issues concluídas — então o SPI essencialmente compara ritmo de entrega de issues com o
cronograma, e o CPI compara esse ritmo com o custo real registrado.
