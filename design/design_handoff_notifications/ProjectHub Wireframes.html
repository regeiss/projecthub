Como ler

Cada frame é uma rota do app. As seções agrupam rotas pela arquitetura de informação.

Símbolos

ação primária / barra preenchida
destaque (selecionado, alerta, hoje)
tracejado = placeholder / opcional

Pontos de destaque

A nav workspace · B nav projeto · C área de conteúdo · D arrastar-p/-status · E arrastar barra p/ reagendar

Fora do escopo (v1)

Módulos, caixa de notificações, busca, mobile — reutilizam o mesmo padrão shell + lista/detalhe.
